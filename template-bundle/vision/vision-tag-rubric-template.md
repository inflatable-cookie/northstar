## Delta markers

- `advanced`
- `neutral`
- `regressed`
- `at_risk`

## Marker expectations

| Marker | Minimum evidence expectation |
| --- | --- |
| `advanced` | linked check/evidence showing concrete movement |
| `neutral` | clear reason movement is preparatory or unchanged |
| `regressed` | explicit issue and containment status |
| `at_risk` | unresolved dependency and named owner |

## Reporting rule

Use the same marker vocabulary in roadmap closures and scorecards.

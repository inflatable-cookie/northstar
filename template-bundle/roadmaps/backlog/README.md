# Roadmap Backlog

Use this folder for deferred items that are not in the active queue.

## Promotion rules

Promote a backlog item when at least one applies:
- it is now required by an active milestone
- repeated requests justify priority change
- dependency timing now makes it tractable

## Authoring rule

Each backlog item should include effort estimate, promotion trigger, and success criteria.
